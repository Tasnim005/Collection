Freezing & Sealing Properties

In JavaScript, you can use Object.freeze() and Object.seal() to control the mutability and configurability of object properties. These methods provide a way to make objects or certain properties read-only and prevent them from being added or removed

Differences:

  • Object.freeze() makes the entire object and all of its properties read-only.
  • Object.seal() allows modifications to existing properties but prevents adding new properties or deleting existing ones.
